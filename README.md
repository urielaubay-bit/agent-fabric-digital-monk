## Cookie consent

The site now includes a lightweight cookie consent manager at `assets/js/cookies.js`.

- Consent is stored in `localStorage` under `dm_cookie_preferences_v1`
- Categories: `necessary` (always on), `analytics`, `marketing`
- Users can reopen preferences from the "Cookie settings" button

To load third-party scripts only after consent, add them as deferred scripts:

```html
<script type="text/plain" data-cookie-category="analytics" src="https://example.com/analytics.js"></script>
```

Use `data-cookie-category="marketing"` for marketing tags.
